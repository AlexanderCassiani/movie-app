# Flix Nova

Flix Nova es una aplicación web tipo catálogo de películas inspirada en plataformas como Netflix. Permite explorar películas, ver detalles y reproducir trailers usando la API de TMDB.

Demo en vivo: https://flix-nova.vercel.app/

---

## Funcionalidades

- Búsqueda de películas
- Listado de películas populares y recomendadas
- Página de detalles de cada película
- Visualización de trailers (cuando están disponibles)
- Modo oscuro / claro
- Diseño responsive adaptado a mobile y desktop

---

## Tecnologías usadas

- React
- Vite
- React Router
- CSS puro (sin frameworks)
- TMDB API (The Movie Database)
- Vercel (deploy)
